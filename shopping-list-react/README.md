# Lista de Compras (React)

Projeto simples de lista de compras/ desejos em React + Vite.

Funcionalidades do MVP:

- Adicionar item com nome, quantidade, preço, categoria e notas
- Listar itens por status (Desejos / Adquiridos)
- Marcar item como comprado (toggle)
- Remover item
- Cálculo de totais por lista
- Persistência local usando `localStorage`

Como rodar (PowerShell):

```powershell
cd c:\Users\Maxsuel\OneDrive\Documentos\html-css-js\HTML-CSS-JS\projeto_teste\shopping-list-react
npm install
npm run dev
```

Observações:

- Este scaffold usa Vite. Se preferir, posso adaptar para Create React App.
- Próximos passos sugeridos: editar item, filtros, busca, arrastar entre listas, export/import JSON, sincronização com backend.
